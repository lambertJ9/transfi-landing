TRANSFI LANDING PAGE SETUP

1. Install Node.js
2. Open terminal in this folder
3. Run:
   npm install
4. Run:
   npm run dev
5. Open:
   http://localhost:3000

UPLOAD TO GITHUB
1. Create GitHub repo: transfi-landing
2. Upload all files from this folder

DEPLOY TO VERCEL
1. Sign in to Vercel with GitHub
2. Import repo
3. Click Deploy

GODADDY DNS
A     @      76.76.21.21
CNAME www    cname.vercel-dns.com

IN VERCEL ADD DOMAINS
- transfi.app
- www.transfi.app
Set www as primary
